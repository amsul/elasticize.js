elasticize.js

dynamically resize textarea height


A tiny (1kb) jQuery plugin to dynamically resize the height of a <textarea> field as you type.

border-width has to be 0 to work right. You can create a fake border with box-shadow like in the demo to the left.

Works best with latest browsers.. **probably not for everyone**

How to use

$('textarea').elasticize();

elasticize.js

dynamically resize textarea height


A tiny jQuery plugin to dynamically resize the height of a <textarea> field as you type.

Works on all major browsers: Chrome, Firefox 3.5+, Safari, IE 8+, Opera 10.5+, iOS 3.2+, Android browser 2.1+

How to use

$('textarea').elasticize();
